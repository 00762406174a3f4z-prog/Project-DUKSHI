# デュクシ - JavaScriptゲーム

## 🎮 ゲーム概要

### ルール
デュクシは、攻撃・防御・チャージの3つのアクションで対戦する、駆け引きが重要なゲームです。

**プレイヤーの初期ステータス:**
- ライフ: 3
- MP（技ポイント）: 3（最大）

**勝利条件:**
- 相手のライフを0にする

## 🛠️ ゲームシステム

### 4つのクラス実装

1. **PlayerStats** - ステータス管理（プレイヤー・ボット共通）
	- ライフ、MP、アクション実行判定

2. **BotAI** - ボット思考エンジン（Easy/Normal/Hard）
	- 難易度別の行動判定AI

3. **GameJudge** - アクション相性判定
	- 攻撃、防御、チャージの相互作用を計算

4. **DukshiGame** - メインゲームロジック
	- UI管理、ターン処理、ゲーム状態管理

## 🎯 ゲームプレイ

### 3つのアクション

👊 **デュクシ（攻撃）** - MP-1、相手ライフ-1
🛡️ **バリア（防御）** - MP-1、相手攻撃を防ぐ
⚡ **チャージ（回復）** - MP+1、無防備（リスク大）

### 基本ルール

- MP0でチャージ強制
- チャージ中に攻撃受けると直撃
- 相手ライフを0にしたら勝利
