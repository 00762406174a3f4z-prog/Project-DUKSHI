# デュクシ - JavaScriptゲーム

## 🎮 ゲーム概要

### ルール
デュクシは、攻撃・防御・チャージの3つのアクションで対戦する、駆け引きが重要なゲームです。

**プレイヤーの初期ステータス:**
- ライフ: 3
- MP（技ポイント）: 3（最大）

**勝利条件:**
- 相手のライフを0にする

## 🛠️ ゲームシステム

### 3つの主要クラス実装

1. **Character** - キャラクター管理
	- ライフ、MP、物理移動、アクション（攻撃、防御、チャージ）の実行。
	- メガデュクシなどの強力な技も管理。

2. **AIBot** - ボット思考エンジン
	- 状況（距離、MP、相手の状態）に応じた動的な意思決定。
	- 難易度（Easy/Normal/Hard）に応じた反応速度の変化。

3. **DukshiGame** - メインゲームロジック
	- Canvas描画、入力管理、衝突判定、UI更新、ゲーム状態管理。

## 🎯 ゲームプレイ

### 3つのアクション

👊 **デュクシ（通常攻撃）** - MP-1、威力1。
💥 **メガデュクシ（必殺技）** - MP-3、威力3。広範囲。
🛡️ **バリア（防御）** - MP消費なし。通常攻撃を無効化、メガ攻撃を大幅軽減。
⚡ **チャージ（回復）** - 無防備な状態でエネルギーを溜め、MPを回復。

### 基本ルール

- 攻撃にはMPが必要。
- チャージ中は移動不可。
- 相手のライフ（3ポイント）を0にしたら勝利。

## 🤖 難易度切り替え

`game.js` の `DukshiGame` クラスのコンストラクタ内で変更可能です：
```javascript
this.ai = new AIBot(this.bot, 'normal'); // 'easy', 'normal', 'hard'
```
